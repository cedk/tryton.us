Open Source



Tryton abides by the GPL license, which ensures that the modifications made are available to every member of the open source community. Being developed by a non-profit foundation which supports complete transparency.

Mature and Quality Code



The code building of Tryton applications is unique in its own way. The code is posted on the open source community to be reviewed by every member, which leaves no stone unturned in its development. After a painstakingly accurate review, the code is merged into the mainstream repositories.

Predictable Release Cycle


Tryton releases a new series every six months and new versions of older releases are made after the bug fixes (if there are any).
