Open Source

Tryton abides by the GPL license, which ensures that the modifications made are available to every member of the open source community. Being developed by a non-profit foundation which supports complete transparency.

MATURE AND QUALITY CODE

The code building of Tryton applications is unique in its own way. The code is posted on the open source community to be reviewed by every member, which leaves no stone unturned in its development. After a painstakingly accurate review, the code is merged into the mainstream repositories

Tryton has one of the active Open source communities in the world.

Tryton project is a global collaboration of developers and technology enthusiasts developing the open source business management software for SMB’s. Backed by a Not for profit Tryton software foundation and vibrant community of developers.

Tryton is a three-tiers high-level general purpose application platform written in Python and use Postgresql as database engine.

Most active open source community, get instant response on mailing list, IRC channels on freenode.

Join Tryton Unconference that happens every year in the month of November and explore creativity with like minded people.
